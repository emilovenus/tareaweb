Hola desde el backend Netlify Frontend · Render Backend

Página de ejemplo para demostrar la conexión entre un frontend estático en Netlify y un backend en Node.js desplegado en Render.

Al hacer clic en el botón, el frontend envía una petición fetch() al servidor backend y muestra la respuesta en tiempo real.

Esperando interacción…
Frontend: Netlify
Backend: Render · Node + Express
Endpoint: /api/mensaje

el boton ejecuta una peticion fetch() hacia:
https://emiweb.onrender.com/api/mensaje y coloca el texto de la respuesta en la sección “Mensaje”.
